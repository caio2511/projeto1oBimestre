Maratona de Programação

A maratona de programação do técnico de informática consiste em um evento básicamente para melhorar e treinar a lógica de programação dos alunos. Podendo escolher alguma das línguagens de programação para resolver os problemas de lógica.

Além de tudo o aluno que participar pode acrescentar isto ao seu currículo que já é um bom começo para alguém que não possui muito expêriencia.
